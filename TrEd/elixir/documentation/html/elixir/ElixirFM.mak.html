ElixirFM - Context for Annotation of the ElixirFM Lexicon in the TrEd Environment

DESCRIPTION

back

ElixirFM is a high-level implementation of Functional Arabic Morphology documented at http://sourceforge.net/p/elixir-fm/wiki/. The core of ElixirFM is written in Haskell, while interfaces in Perl support lexicon editing and other interactions.

TrEd Tree Editor http://ufal.mff.cuni.cz/tred/

Prague Arabic Dependency Treebank http://ufal.mff.cuni.cz/padt/online/
